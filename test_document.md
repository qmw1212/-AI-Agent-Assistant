# AI Agent Assistant - 测试文档

这是一个用于测试文件分析工具的示例文档。

## 项目简介

AI Agent Assistant是一个智能多工具代理系统，具备以下核心能力：

1. **智能推理**: 使用ReAct框架进行思考和行动
2. **工具调用**: 自动选择和使用合适的工具
3. **记忆系统**: 记住对话历史和用户信息
4. **现代UI**: 基于Streamlit的友好界面

## 技术特点

- 使用LangChain构建Agent
- 集成Google Gemini LLM
- 支持多种工具扩展
- 实时推理过程可视化

## 使用场景

1. 信息检索助手
2. 数学计算助手
3. 文档分析助手
4. 个人知识管理

## 测试说明

你可以使用以下命令测试文件分析功能：

```
用户: 帮我分析 test_document.md 文件
```

Agent将会：
1. 读取文件内容
2. 统计字数、行数
3. 显示内容预览
4. 提供文件摘要
